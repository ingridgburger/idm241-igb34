Beta Description

Anthropologie Product Image Card

IDM 241 - Ingrid Burger

https://www.anthropologie.com/dresses

Triggers

  • Object 1 - Product Image
    • Hover On
    • Hover Off
    • Click
  • Object 2 - Heart Button
    • Hover On
    • Hover Off
    • Click
  • Object 3 - Color Swatch Button
    • Hover On
    • Hover Off
    • Click
  • Object 4 - Badge (New/Top Rated/Plus Sizes)
    • Card Hover On
    • Card Hover Off
  • Object 5 - Image Gallery Navigation (Dots & Arrows)
    • Hover On
    • Hover Off
    • Click

Rules

  • Object 1 - Product Image
    • Hover On
      • Cursor changes to pointer
      • Default image fades out, hover image fades in
      • Product image scales up to 1.15x (contained within container)
      • Quickshop bar slides up from bottom
      • Heart button scales in at top-right
      • Product name text turns blue
      • Gallery arrows fade in
    • Hover Off
      • Cursor reverts to default
      • Hover image fades out, default image fades in
      • Image scales back to 1x
      • Quickshop bar slides back down
      • Heart button scales out
      • Product name text reverts to black
      • Gallery arrows fade out
    • Click
      • Navigates to product detail page
  • Object 2 - Heart Button
    • Hover On
      • Cursor changes to pointer
      • Button scales up slightly (1.05x)
    • Hover Off
      • Cursor reverts to default
      • Button returns to original scale
    • Click
      • Button scales up to 1.2x quickly, then back to 1x after animation completes
      • Background changes from white to blue
      • Heart outline stays white and fills bottom-to-top with white color
      • After fill completes, button scales back down to 1x
      • State persists (clicking again reverses the fill and returns background to white, outline to blue)
  • Object 3 - Color Swatch Button
    • Hover On
      • Cursor changes to a finger.
      • Blue outline grows out around color swatch.
    • Hover Off
      • Cursor reverts to default
      • If swatch not selected, outline reverts to black
      • If swatch selected, black outline remains
    • Click
      • Blue outline around color swatch changes to black outline.
        • Blue ripple emanates from click point, expands and fades
        • Swatch becomes selected with black outline, previous swatch deselects
        • Product images fade out, update to new color, fade in
        • Gallery resets to first image
  • Object 4 - Badge (New/Top Rated/Plus Sizes)
    • Card Hover On
      • Badge rotates 5-10 degrees clockwise and scales to 1.1x
      • Text may pulse once
      • Badge always visible in top-left corner with blue or white background and black text
    • Card Hover Off
      • Badge rotates back to original position
      • Badge scales back to 1x
  • Object 5 - Image Gallery Navigation (Dots & Arrows)
    • Hover On
      • Cursor changes to pointer
      • Arrow becomes fully opaque and scales to 1.1x
    • Hover Off
      • Cursor reverts to default
      • Arrow returns to semi-transparent state and original scale
    • Click
      • Arrow has press animation (scales to 0.95x then back)
      • Current image fades out, next/previous image fades in (~300ms)
      • Arrows only appear with 2+ images
      • Arrows positioned on left/right edges of image, vertically centered
      • Circular buttons (~30-35px) with semi-transparent white background
      • Simple chevron icons (< and >)
      • Left arrow hidden on first image, right arrow hidden on last image
