Anthropologie Product Image Card

IDM 241 - Ingrid Burger

Triggers

  • Object 1 - Product Image
    • Hover On
    • Hover Off
    • Click
  • Object 2 - Heart Button
    • Hover On
    • Hover Off
    • Click
  • Object 3 - Color Swatch Button
    • Hover On
    • Hover Off
    • Click
  • Object 4 - Badge
    • Card Hover On
    • Card Hover Off
  • Object 5 - Image Gallery Navigation
    • Hover On
    • Hover Off
    • Click

Rules

  • Object 1 - Product Image
    • Hover On
      • Cursor changes to a pointer when hovering over the product card.
      • The current product image scales up from 1x to 1.15x over 0.45 seconds.
      • The Quickshop bar slides up into view from the bottom of the image over 0.3 seconds.
      • The product name text color transitions from black to blue over 0.3 seconds.
      • The badge in the top-left corner rotates slightly and scales up over 0.2 seconds.
      • The gallery dots move upward so they appear above the Quickshop bar over 0.3 seconds.
      • The gallery arrows remain visible on the left and right edges of the image (left arrow may be hidden on the first image, right arrow on the last image).
    • Hover Off
      • Cursor returns to the default arrow.
      • The product image scale returns from 1.15x back to 1x over 0.45 seconds.
      • The Quickshop bar slides back down and out of view over 0.3 seconds.
      • The product name text color transitions from blue back to black over 0.3 seconds.
      • The badge rotates back to its original upright position and returns to 1x scale over 0.2 seconds.
      • The gallery dots move back down toward the bottom edge of the image over 0.3 seconds.
    • Click
      • Clicking anywhere on the product card that is not a button (arrows, heart, or color swatches) opens the product detail page in a new browser tab.
  • Object 2 - Heart Button
    • Hover On
      • Cursor changes to a pointer when hovering over the heart button.
      • The heart button scales up slightly from 1x to 1.05x over 0.2 seconds.
      • The heart icon color transitions from black to blue over 0.2 to 0.3 seconds.
    • Hover Off
      • Cursor returns to the default arrow.
      • The heart button returns from 1.05x back to its original 1x scale over 0.2 seconds.
      • If the heart is not saved, the icon color transitions back from blue to black over 0.2 to 0.3 seconds.
    • Click
      • When clicked, the heart button briefly scales up more and then settles back to its normal scale over 0.2 seconds.
      • The heart button toggles a saved state using a CSS class.
      • In the saved state, the border color transitions from white to blue over 0.3 seconds.
      • In the saved state, a blue fill inside the circular button rises upward from the bottom to the top over 0.3 seconds.
      • In the saved state, the heart icon color changes to white so it appears as a white heart on top of the blue fill.
      • Clicking the heart again removes the saved state: the blue fill animates from 100% height back to 0% height inside the circle over about 0.3 seconds, revealing the white background again.
  • Object 3 - Color Swatch Button
    • Hover On
      • Cursor changes to a pointer when hovering over a color swatch.
      • The swatch scales up from 1x to about 1.05x over about 0.2 seconds.
      • For an unselected swatch, the outline transitions to a blue hover outline over 0.2 seconds.
    • Hover Off
      • Cursor returns to the default arrow.
      • For an unselected swatch, the outline transitions back from the blue hover outline to the default black outline and the swatch returns to 1x scale over 0.2 seconds.
      • The selected swatch keeps its thicker blue outline and slightly larger scale even when the cursor moves away.
    • Click
      • The clicked swatch becomes selected, gaining a thicker blue outline and a slightly larger scale over 0.2 seconds.
      • The previously selected swatch is deselected and returns to the default black outline and normal scale.
      • The product gallery resets to the first image for the newly selected color.
      • The gallery image updates by crossfading from the previous image to the new image using an overlay that fades out over 0.4 seconds, while the underlying images adjust their opacity over 0.45 seconds.
      • The gallery dots update to indicate the new current image position.
  • Object 4 - Badge
    • Card Hover On
      • The circular badge remains visible in the top-left corner of the product image at all times.
      • When the card is hovered, the badge rotates slightly and scales up over 0.2 seconds.
      • The badge background remains blue with white text during the interaction.
    • Card Hover Off
      • When the card is no longer hovered, the badge rotates back to its original upright position over 0.2 seconds.
      • The badge scale returns from its larger hover state back to 1x over 0.2 seconds.
      • The badge remains visible in the top-left corner before, during, and after hover.
  • Object 5 - Image Gallery Navigation
    • Hover On
      • Cursor changes to a pointer when hovering over the gallery arrows.
      • The arrow buttons scale up slightly over 0.2 seconds.
      • The arrow border and icon color transition from black to blue over 0.3 seconds.
    • Hover Off
      • Cursor returns to the default arrow.
      • The arrow buttons return to their original scale over 0.2 seconds.
      • The arrow border and icon color transition back from blue to black over 0.3 seconds on a white circular background.
    • Click
      • When an arrow is clicked, it briefly scales up more and then returns to its normal size over about 0.2 seconds.
      • Clicking an arrow updates the gallery position to the next or previous image.
      • The current image crossfades into the new image using an overlay that fades out over 0.4 seconds while the underlying images update and apply their own opacity transitions of 0.45 seconds.
      • The left arrow is hidden (opacity 0 and no pointer events) when there is no previous image; the right arrow is hidden when there is no next image.
      • The gallery dots update so the active dot always matches the current image in the sequence.
