Beta Description

Anthropologie Product Image Card

IDM 241 - Ingrid Burger

Triggers

  • Object 1 - Product Image
    • Hover On
    • Hover Off
    • Click
  • Object 2 - Heart Button
    • Hover On
    • Hover Off
    • Click
  • Object 3 - Color Swatch Button
    • Hover On
    • Hover Off
    • Click

Rules

  • Object 1 - Product Image
    • Hover On
      • Cursor changes to a finger.
      • Product image fades to secondary image.
      • Quickshop bar appears slides in from bottom of the image:
        • White bar with 80% opacity.
        • Text reading "Quickshop" centered.
        • Covers 100% of the width and 10% of the height.
      • Small heart button scales in at the top-right of the image:
        • Approximately 20px x 20px.
        • Positioned with 15px margin from top and right edges.
        • Circle with 80% opacity and a heart outline centered.
      • Product name text instantly changes color to blue.
    • Hover Off
      • Cursor reverts to default.
      • Product image reverts to default.
      • Quickshop bar slides out to bottom.
      • Heart button scales out.
      • Product name text color fades to revert to default.
    • Click
      • Clicking the product image navigates the user to the product detail page.
  • Object 2 - Heart Button
    • Hover On
      • Cursor changes to a finger.
      • Outline of heart changes from black to blue.
    • Hover Off
      • Cursor reverts to default.
      • Outline of heart changes from blue to black.
    • Click
      • The heart outline animates as blue color fills it from the bottom upward, creating a smooth vertical fill effect.
  • Object 3 - Color Swatch Button
    • Hover On
      • Cursor changes to a finger.
      • Blue outline draws a circle around color swatch.
    • Hover Off
      • Cursor reverts to default.
      • If color swatch not selected, blue outline around color swatch changes from blue to black.
      • If color swatch already selected, black outline around color swatch remains the same.
    • Click
      • Blue outline around color swatch changes to black outline.
        • Black outline draws a circle around color swatch.
