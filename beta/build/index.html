Anthropologie Product Image Card

IDM 241 - Ingrid Burger

Triggers

  • Object 1 - Product Image
    • Hover On
    • Hover Off
    • Click
  • Object 2 - Heart Button
    • Hover On
    • Hover Off
    • Click
  • Object 3 - Color Swatch Button
    • Hover On
    • Hover Off
    • Click

Rules

  • Object 1 - Product Image
    • Hover On
      • Cursor changes to a pointer.
      • Default product image fades out, hover image fades in.
      • Quickshop bar slides up from bottom of image.
      • Heart button scales in at top-right of image.
      • Product name text color changes to blue.
    • Hover Off
      • Cursor reverts to default.
      • Hover image fades out, default image fades in.
      • Quickshop bar slides back down.
      • Heart button scales out.
      • Product name text color reverts to default.
    • Click
      • Clicking the product image navigates to the product detail page.
  • Object 2 - Heart Button
    • Hover On
      • Cursor changes to a pointer.
      • Heart outline color changes from black to blue.
    • Hover Off
      • Cursor reverts to default.
      • Heart outline color reverts from blue to black.
    • Click
      • Heart outline toggles fill animation, filling blue from bottom upward.
  • Object 3 - Color Swatch Button
    • Hover On
      • Cursor changes to a pointer.
      • Blue outline grows out to appear around swatch.
    • Hover Off
      • Cursor reverts to default.
      • If swatch not selected, outline reverts to black.
      • If swatch selected, black outline remains.
    • Click
      • Swatch becomes selected, black outline appears.
      • Previously selected swatch is deselected.
      • Product images update according to selected swatch.
