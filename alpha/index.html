Anthropologie Product Image Microinteraction

IDM 241 - Ingrid Burger

Original Interface URL

https://www.anthropologie.com/

Triggers

  • User hovers over the product image or product name.
  • User clicks on the product image or product name.

Rules

  • Hover On
    • Cursor changes to a finger.
    • Product image instantly changes to secondary image.
    • Quickshop bar appears instantly at the bottom of the image:
      • White bar with 80% opacity.
      • Paragraph-size text reading "Quickshop" centered.
      • Covers 100% of the width and 10% of the height.
    • Small heart button appears instantly at the top-right of the image:
      • Approximately 20px x 20px.
      • Positioned with 15px margin from top and right edges.
      • Circle with 80% opacity and a heart outline centered.
    • Product name text instantly changes color to blue.
  • Hover Off
    • Cursor changes back to default.
    • Product image reverts to default.
    • Quickshop bar disappears.
    • Heart button disappears.
    • Product name text returns to its default color.
  • Click
    • Clicking the product image navigates the user to the product detail page.

Feedback

  • Visual
    • Cursor changes to pointer.
    • Product images changes to secondary image.
    • Quickshop bar and heart button appear.
    • On click, the user navigates to the product detail page.
  • Audio
    • None
  • Feeling / User Perception
    • User sees interactivity and responsiveness of the item.
    • Visual changes provide confirmation that hover and click actions are recognized.
  • User feels a sense of interactivity and confirmation from the click.

Loops & Modes

  • The microinteraction does not repeat automatically.
  • Hover and click function independently.
